FTP Server Example

The NetBurner module is running a FTP Server that enables you to download a text file containing the number of seconds since boot. To obtain the data file:

  1. Run a FTP client program on your host computer. You can do this easily with most operating system platforms by opening a command prompt window and typing "ftp <ip address>, where the <ip address> is the address of the NetBurner.
  2. Once connected, hit the <enter> key when asked for user name and password (this example does not use authentication)
  3. To view the downloadable files, type "ls". You should see "seconds.txt".
  4. Use the FTP "get <file name>" command to download files. For the seconds since boot, type "get seconds.txt".
  5. Now you can view the file on your local hard drive.
  6. To exit the FTP host program, type "quit" and press <enter>.
